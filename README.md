🏏 IPL Analytics Dashboard

📌 Project Overview

This Power BI project analyzes IPL match data to provide insights into team performance, runs scored, wickets taken, and match statistics.

✔ Team Performance  
✔ Runs Analysis  
✔ Wickets Analysis  
✔ Match Statistics  
✔ Team Comparison  

📊 Dashboard Features

- Total Runs KPI
- Total Wickets KPI
- Runs by Teams Analysis
- Wickets by Teams Analysis
- Team Status Distribution
- Runs and Wickets Comparison
- Interactive Team Filters
- Match-wise Statistics

🛠 Tools Used

- Power BI
- Power Query
- DAX
- Data Cleaning
- Dashboard Design

📈 Key Insights

- Highest Scoring Team Analysis
- Best Wicket Taking Team Analysis
- Team Performance Comparison
- Match Statistics Tracking
- Interactive Reporting
